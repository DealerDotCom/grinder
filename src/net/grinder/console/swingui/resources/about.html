The Grinder is a pure Java load-testing framework released under the GNU Public License. It was originally developed for the book "Professional Java 2 Enterprise Edition with BEA WebLogic Server" by Paco Gómez and Peter Zadrozny.

Philip Aston took ownership of the code and reworked it to create The Grinder 2. Philip continues to enhance and maintain The Grinder.

The Grinder's home page is currently http://grinder.sourceforge.net/.
