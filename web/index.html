What is The Grinder?

The Grinder is a pure Java load-testing framework that is freely available under a BSD style open-source license. It was originally developed for the book Professional Java 2 Enterprise Edition with BEA WebLogic Server by Paco Gómez and Peter Zadrozny.

Philip Aston took ownership of the code and reworked it to create The Grinder 2. Philip continues to enhance and maintain The Grinder, and welcomes all contributions.

The Grinder's home page is http://grinder.sourceforge.net/.
